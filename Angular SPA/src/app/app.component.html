Origen del proyecto

Esta aplicación fue desarrollada como proyecto para el curso de Paradigmas de Programación - II Ciclo 2019. El objetivo principal fue aplicar técnicas de programación LP-FP, para desarrollar un problema computacional no tradicional: un compilador e interprete, de cierta parte del DSL Rivescript.
